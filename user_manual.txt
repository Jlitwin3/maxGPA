MaxGPA User Documentation

--------------------------------------------------
Overview
--------------------------------------------------

MaxGPA is an interactive planning tool that helps students build a graduation plan while selecting instructors strategically. The system allows users to:

- Select a major and time window
- Generate a multi-year course schedule
- Add required courses to specific terms
- Choose instructors for each course
- View grade distribution data to support decisions

The system guides the user through building a complete graduation plan step by step.

--------------------------------------------------
Getting Started
--------------------------------------------------

1. Select User Type

When the application loads, you will see two buttons:
- Student
- Admin

Click "Student" to begin building a graduation plan.

--------------------------------------------------
Task 1: Generate a Graduation Plan
--------------------------------------------------

Goal:
Create an empty schedule based on your major and timeline.

Steps:
1. Select a Major from the dropdown menu.
2. Select a Start Year.
3. Select an End Year.
4. Click "Generate Plan".

Result:
A grid will appear showing:
- Rows = Academic years (AY16, AY17, etc.)
- Columns = Terms (Fall, Winter, Spring, Summer)
- Each cell represents a term where courses can be added.

--------------------------------------------------
Task 2: Add a Course to a Term
--------------------------------------------------

Goal:
Place a required course into a specific term.

Steps:
1. Click on any cell in the grid.
2. In the Course Selection section below:
   - Open the "Add Course" dropdown.
   - Select a course.

Result:
- The course appears in the selected cell as a pending course.
- The system prevents duplicate course selection across the plan.

--------------------------------------------------
Task 3: Choose an Instructor
--------------------------------------------------

Goal:
Select an instructor for the chosen course.

Steps:
1. After selecting a course, look to the right of the grid.
2. In the Instructor Selection panel:
   - Review available instructors.
   - Each instructor shows grade distribution data (A, B, C, DNF).
3. Select an instructor using the radio button.

Result:
- The selected instructor is associated with the course.
- The Course Selection panel updates to reflect your choice.

--------------------------------------------------
Task 4: Lock in a Course
--------------------------------------------------

Goal:
Finalize a course and instructor selection.

Steps:
1. Ensure both:
   - A course is selected
   - An instructor is selected
2. Click "Lock In Class"

Result:
- The course is permanently added to the cell.
- The instructor is saved with that course.
- Each cell can contain up to 4 classes.

--------------------------------------------------
Task 5: Modify a Term
--------------------------------------------------

Cancel Current Course:
- Click "Cancel Current Class"
- Removes the pending (unlocked) course

Clear Entire Cell:
- Click "Clear Cell"
- Removes all courses (locked and pending) from the term

--------------------------------------------------
Task 6: View Grade Distribution Panel
--------------------------------------------------

Goal:
Access grade distribution information.

Steps:
- Use the "Collapse Graph Panel" / "Open Graph Panel" button

Result:
- Shows or hides the Grade Distribution panel
- This panel displays contextual information for the selected course

Note:
Instructor selection is NOT in this panel. It is always located to the right of the grid.

--------------------------------------------------
System Rules and Constraints
--------------------------------------------------

- Each term (cell) can contain up to 4 courses
- Courses cannot be duplicated across the schedule
- A course must have an instructor before being locked
- Only required courses for the selected major are available

--------------------------------------------------
Troubleshooting
--------------------------------------------------

Problem: Cannot lock in class
Solution:
- Ensure both a course and an instructor are selected

Problem: No courses available in dropdown
Solution:
- All required courses may already be scheduled

Problem: Instructor panel is empty
Solution:
- Click a grid cell and select a course

Problem: Grid not appearing
Solution:
- Select a major, start year, and end year, then click "Generate Plan"

--------------------------------------------------
Summary of Workflow
--------------------------------------------------

1. Select Student
2. Choose Major and Years
3. Generate plan
4. Click a cell
5. Select a course
6. Choose instructor (right panel)
7. Lock in class
8. Repeat until plan is complete
